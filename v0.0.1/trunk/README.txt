Block malicous IP's from our database of known botnets, malware, hackers and c&c servers.

== Description ==

We will add to the list in real time, as well as with your permission collect IP's to block from malicious requests to your website.

== Installation ==

Search for wpblock in the Wordpress plugin repository and click install. Or you can download from http://wpblock.io

== Frequently Asked Questions ==

= Where do you get the IPs to blacklist? =

From various well known blacklists, malware analysis, industry newsletters, honeypot logs and request analysis.
